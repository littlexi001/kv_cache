## Metric Notes

- `A` 表示 same-input-different-output slot；`B` 表示 different-input-same-output slot。
- `Output-token acc` 只统计 slot 中倒数第二个位置预测最后一个 output symbol 的准确率。
- `A/B group same-expert` 是同一结构 group 内 token 被分到同一 expert bucket 的比例；dense 模型没有该项。
- `Attn A/B mass` 是 attention mass 落在同一 A/B group 内 token 上的比例，包含 self attention。

## Current Research Takeaway

The viable direction is no longer "replace all remote KV with synthetic prototypes". The more promising version is:

**Layer-Selective Influence-Bounded Synthetic KV**

For each layer, fit synthetic KV prototypes to approximate the attention output on calibration queries. Then select only layers whose functional replacement is low-influence under a layer gate. Sensitive layers keep real KV; safe layers use synthetic KV.

This gives a clearer boundary from SparQ, Quest, and DuoAttention:

- SparQ/Quest style: token/head selection from existing KV.
- DuoAttention style: split heads/layers by attention pattern.
- This project: approximate the attention function with learned synthetic KV prototypes, then bound the deployment by layer influence.
